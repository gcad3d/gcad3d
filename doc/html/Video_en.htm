How to create gcad3d-tutorial-videos



____________________________________________________________________________
Prerequisites:
- install simplescreenrecorder

Start gcad
  - activate Options / ScreenCast ON

  simplescreenrecorder
  - audio off;
  - select the window you want to record
  - Weiter
  - ~/.mp4
  - Container mp4
  - Codec H.264
  - Weiter
  - Aufnahme starten - start recording
  - - im Aufnahmefenster agieren -
  - Aufnahme speichern - stop (select the icon (red square) on top
  - wait for the output file ~/.mp4
